*   Problem 513

   ABC is an integral sided triangle with sides a≤b≤c.
   m_c is the median connecting C and the midpoint of AB.
   F(n) is the number of such triangles with c≤n for which m_c has integral
   length as well.
   F(10)=3 and F(50)=165.

   Find F(100000).
